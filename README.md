## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

#### ‼️ 위에서도 설명했지만 'npm run dev' 다. 누군지 굳이 언급 안하겠지만 나한테 "우웅 pull 했는데 왜 안되징...?" 하던애 때문에 강조한다. 

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
esLint 썻다 참고해라.
```
